== Description ==

A clean admin dashboard for your vehicle and bus data. See stats, recent updates, and prepare for advanced analytics. Works great alongside your custom importers and Vehicle Info plugins.

== Installation ==

1. Upload the plugin files to /wp-content/plugins/vehicle-data-dashboard
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to the new 'Vehicle Dashboard' menu item
