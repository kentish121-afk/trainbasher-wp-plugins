== Description ==

Automatically outputs rich Vehicle schema markup (JSON-LD) on your vehicle posts. Improves SEO and rich results in Google. Easily extensible for your custom fields and works alongside Yoast SEO.

== Installation ==

1. Upload to /wp-content/plugins/seo-schema-enhancer-vehicles
2. Activate
3. Schema is added automatically on vehicle singular pages. Customize post types via filter.

== Frequently Asked Questions ==

= How do I change which post types get schema? =
Use the `vse_vehicle_post_types` filter in your theme or another plugin.
