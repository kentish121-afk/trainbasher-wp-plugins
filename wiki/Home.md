# TrainBasher WP Plugins Wiki

Welcome to the official documentation for the custom WordPress plugins built for [trainbasher.com](https://trainbasher.com/) — the UK bus photography archive.

These plugins were developed to enhance the site for bus enthusiasts and photographers by adding powerful search, structured data, and insightful statistics.

## Available Plugins

- **[Bus Search Pro](Bus-Search-Pro)** — Advanced searchable database with filters, migration tool, and structured sighting data.
- **[Collection Stats Dashboard](Collection-Stats-Dashboard)** — Beautiful stats, charts, leaderboards, and public shortcode.

## Quick Start

1. Clone or download this repository.
2. Upload the desired plugin folder(s) to `wp-content/plugins/`.
3. Activate in WordPress.
4. Run the **Migration Tool** (in Bus Search settings) for best results on existing content.
5. Add the shortcodes to pages as needed.

## Key Features Across Plugins

- Shared custom post meta fields for consistency
- AJAX-powered interfaces
- Mobile-friendly and performant
- Designed to work with the Newsup theme (and most others)
